

好问题，这是核心中的核心。成就系统如果设计不好，只是把"赚币"换成了"集徽章"，换汤不换药。

关键在于：**不是量化成就本身，而是量化成就背后的行为模式变化。**

---

**自驱力的可量化信号（行为指标）：**

**1. 主动性指标 - "谁发起的？"**
```
主动完成率 = 无提醒完成次数 / 总完成次数

数据采集方式：
- 任务完成时记录"触发方式"：
  * 孩子自己点击完成（主动）
  * 家长提醒后完成（被动）
  * 临近deadline才完成（拖延）
```

自驱力提升的信号：主动完成率从30%→60%→80%

**2. 探索性指标 - "愿不愿意尝试新东西？"**
```
探索广度 = 尝试过的任务类型数 / 可用任务类型总数
探索深度 = 非必选任务完成次数 / 必选任务完成次数

数据采集：
- 标记任务属性：必选 vs 可选、熟悉 vs 新领域
- 记录孩子首次尝试新类型任务的时间点
```

自驱力提升的信号：孩子开始主动尝试"没有高额奖励但有趣"的任务

**3. 持续性指标 - "停止奖励后还做吗？"**

这是最关键的验证。可以设计一个实验：
```
A/B测试：
- 某类任务前3周：每次10币
- 第4周开始：降为5币
- 观察完成率变化

如果完成率不降反升 → 说明已经形成习惯，不依赖奖励
如果完成率断崖下跌 → 说明还是外驱为主
```

**4. 质量指标 - "是应付还是真学？"**
```
深度参与度 = 超额完成次数 / 总完成次数

例子：
- 任务要求"阅读30分钟"，孩子读了50分钟
- 任务要求"背10个单词"，孩子主动背了20个
- 任务要求"画一幅画"，孩子画完还主动写了创作说明
```

**成就系统的反设计 - 让成就"不可追"**

传统成就系统的问题：孩子看到"连续7天学习"的徽章，就会为了徽章而学，本质还是外驱。

**解决方案：延迟揭晓 + 意外惊喜**

```
设计思路：
1. 成就条件不透明 - 孩子不知道做什么能解锁
2. 成就触发后才显示 - "恭喜你解锁了'好奇宝宝'成就！
   系统发现你最近3周主动尝试了5种不同类型的任务"
3. 成就有故事而非数字 - 不是"完成100次任务"，
   而是"你已经连续2周每天都自己想起来学习，
   没有爸妈提醒，这说明学习正在变成你的习惯"
```

这样设计的好处：
- 孩子无法"刷"成就
- 成就变成对行为的"发现"和"命名"，而不是目标
- 强化的是过程（"你很主动"）而非结果（"你赚了很多币"）

**量化自驱力的综合模型：自驱力指数（SDI - Self-Drive Index）**

```python
SDI = (
    主动性得分 × 0.4 +      # 最重要
    探索性得分 × 0.2 +      
    持续性得分 × 0.2 +      
    质量得分 × 0.2
) × 时间衰减系数

时间衰减系数：
- 近7天的行为权重 = 1.0
- 8-14天的行为权重 = 0.7
- 15-30天的行为权重 = 0.4
```

**关键：这个指数只给家长看，不给孩子看。**

家长端显示：
```
小明的自驱力指数：68分（上周62分 ↑）

分析：
✅ 主动性提升明显：本周有5次是自己想起来学习的
⚠️ 探索性还需加强：最近2周只尝试了数学和英语，
   建议引导他试试科学实验类任务
📈 趋势：过去4周自驱力稳步上升，说明激励策略有效
```
